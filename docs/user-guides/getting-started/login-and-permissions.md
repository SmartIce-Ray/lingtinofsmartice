# 登录与权限

## 访问地址

在浏览器中打开 **https://lt.smartice.ai**，推荐使用手机浏览器（支持 iOS Safari 和 Android Chrome）。Lingtin 是一个 PWA 应用，你可以将它添加到手机主屏幕，获得接近原生 App 的体验。

## 登录步骤

1. 打开访问地址，系统自动跳转到登录页面
2. 输入你的**用户名**和**密码**（由管理员提供）
3. 点击「登录」按钮
4. 系统根据你的角色自动跳转到对应页面

> 登录状态会保存在浏览器中，关闭页面后再次打开无需重新登录。如需切换账号，清除浏览器缓存即可。

## 角色与权限

系统目前有两种角色，登录后自动进入对应界面：

| 角色 | 登录后跳转 | 可访问页面 | 数据范围 |
|------|-----------|-----------|----------|
| **店长** | `/recorder` 录音页 | 录音、看板、AI 智库 | 仅本店数据 |
| **管理员**（老板） | `/admin/dashboard` 经营看板 | 经营看板、门店详情、员工提问、问卷管理、AI 智库 | 所有门店数据 |

### 店长能做什么

- **录音**：执行桌访录音和例会录音，AI 自动转写和分析
- **看板**：查看本店的覆盖率、情绪概览、菜品排名、话术评估
- **行动建议**：查看并处理 AI 生成的改善建议
- **AI 智库**：用自然语言向 AI 提问，获取本店经营洞察

### 管理员能做什么

- **经营看板**：一屏纵览所有门店的桌访执行情况和情绪指标
- **门店详情**：深入查看任意门店的桌访记录和顾客反馈原文
- **员工提问**：了解各门店店长的 AI 问答记录和桌访内容
- **问卷管理**：配置桌访问卷模板，指导店长在桌访时应该询问哪些问题
- **AI 智库**：用自然语言提问，获取跨门店的经营分析

## 首次使用提醒

- **店长**：首次录音时，浏览器会弹出麦克风授权请求，请点击「允许」。如果误点了「拒绝」，需要在浏览器设置中手动开启麦克风权限。
- **所有角色**：建议将网页添加到手机主屏幕（iOS: Safari → 分享 → 添加到主屏幕；Android: Chrome → 菜单 → 添加到主屏幕），这样每次打开更方便。
