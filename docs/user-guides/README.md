# Lingtin 产品使用手册

欢迎使用 **Lingtin 语音智能管理平台**——餐饮行业的语音驱动管理工具。

Lingtin 帮助餐饮团队建立一个闭环：**说了 → 记了 → 做了 → 验了**。通过录音采集顾客真实反馈，由 AI 自动分析并生成可执行的改善建议，让每一次桌访都转化为门店进步的动力。

## 选择你的角色

| 角色 | 入口 | 适用人群 |
|------|------|----------|
| [**店长**](store-manager/README.md) | `/recorder` | 店长、值班经理——每天在门店执行桌访、查看数据、处理行动建议 |
| [**管理层**](management/README.md) | `/admin/dashboard` | 老板、区域经理、运营总监——跨门店监控经营状况、管理问卷模板 |
| [**店员**](staff/README.md) | 即将推出 | 服务员、厨师——未来将提供专属功能 |
