# 店长使用手册

## 你的角色

作为店长，你是门店与顾客之间最直接的桥梁。Lingtin 帮助你把每天的桌访从「完成任务」变成「获取洞察」——通过 AI 分析真实的顾客声音，让你清楚地知道哪些做得好、哪些需要改进。

## 三个核心页面

| 页面 | 路径 | 一句话说明 |
|------|------|-----------|
| **录音** | `/recorder` | 桌访录音 + 例会录音，AI 自动转写分析 |
| **看板** | `/dashboard` | 今日数据一览——覆盖率、情绪、菜品反馈、话术评估、行动建议 |
| **AI 智库** | `/chat` | 向 AI 提问，获取经营洞察和建议 |

页面之间通过底部导航栏切换。

## 典型一天的流程

```
──── 午市前 ────
1. 打开 Lingtin，进入录音页 → 切到「例会」标签
2. 选择「餐前会」，录下今天的餐前会
3. 会议结束后查看 AI 生成的会议纪要和行动待办

──── 午市中 ────
4. 切回「桌访」标签，选桌号开始桌访录音
5. 查看问卷提示（如有），按提示与顾客交流
6. 需要时开启隐身模式，让手机看起来像在聊天
7. 每桌录完后查看列表确认状态

──── 午市后 ────
8. 打开看板页，查看午市覆盖率和情绪概览
9. 处理 AI 行动建议——高优先级的建议优先知悉和解决
10. 切到「例会」标签，录下每日复盘会议

──── 晚市重复 4-9 ────

──── 每周 ────
11. 录下周例会
12. 用 AI 智库回顾本周趋势：覆盖率、菜品反馈、情绪变化
```

## 详细指南

按以下顺序阅读，与你的日常工作流程一致：

1. [**桌访录音**](01-table-visit-recording.md) — 选桌号、录音、隐身模式、问卷提示、查看历史
2. [**例会录音**](02-meeting-recording.md) — 餐前会/复盘/周例会、AI 会议纪要
3. [**数据看板**](03-dashboard.md) — 覆盖率、情绪概览、菜品排名、话术评估
4. [**AI 行动建议**](04-action-items.md) — 查看建议、知悉/解决/忽略、查看原始反馈
5. [**AI 智库**](05-ai-chat.md) — 自然语言提问、快捷问题、话术优化
