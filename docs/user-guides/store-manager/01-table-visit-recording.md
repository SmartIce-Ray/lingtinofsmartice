# 桌访录音

## 为什么录音

手写记录容易遗漏细节，事后回忆更不可靠。录音让 AI 捕捉到顾客的每一句话——包括那些你可能觉得「不重要」但实际上隐含着关键信号的细节。你只需要专注和顾客交流，剩下的分析工作交给 AI。

## 前置条件

- 已登录店长账号
- 浏览器已授权麦克风权限（首次录音时会弹出授权请求）

## 操作步骤

### 1. 选择桌号

进入录音页（默认页面），确保顶部标签在「桌访」。点击桌号选择器，选择你即将走访的桌号（如 A1、B3）。

> **必须先选桌号才能开始录音**，这是因为系统需要将录音关联到具体的桌位，才能后续统计覆盖率和按桌追踪反馈。

### 2. 查看问卷提示（如有）

如果管理层配置了当前生效的问卷模板，录音按钮上方会出现一个琥珀色提示卡片，列出今天桌访时应该询问的标准问题。

- 点击卡片可以展开/折叠问题列表
- 这些问题是你的参考提纲，不需要逐字照读——用自然的方式把这些话题融入和顾客的对话中

### 3. 开始录音

点击红色录音按钮，屏幕出现波形动画和计时器（MM:SS 格式）。把手机放在合适的位置，开始与顾客交流。

**录音建议：**
- 单条录音控制在 **1 分钟以内**（约 500KB），过长的录音 AI 分析效果会下降
- 手机距离顾客 30-50cm 效果最佳
- 环境嘈杂时尽量靠近顾客说话的方向

### 4. 使用隐身模式（可选）

录音开始后，屏幕上会出现一个绿色气泡按钮。点击它会切换到一个模拟的聊天界面，让手机看起来像在使用普通聊天软件。

**为什么用隐身模式：** 有些顾客看到录音界面可能会不自在，说话变得拘谨。隐身模式让他们不会注意到手机在录音，从而给出更真实的反馈。

- 点击绿色气泡进入隐身模式
- 点击屏幕任意位置退出隐身模式，返回录音界面
- 隐身模式下录音正常进行，不受影响

### 5. 停止录音

再次点击录音按钮停止。系统会立即提示「A1 桌录音已保存」，然后在后台静默处理：

```
上传音频 (~2秒) → 语音识别 (~3-5秒) → AI 分析 (~7-10秒) → 完成
```

处理完成后会收到提示「A1 桌分析完成」。整个过程约 10-15 秒，期间你可以继续走访下一桌。

### 6. 查看录音历史

录音按钮下方显示今日的录音列表（可切换查看昨日记录）。每条记录显示桌号、时间和处理状态：

| 状态 | 含义 | 操作 |
|------|------|------|
| 已保存 | 存在本地，等待上传 | 自动处理，无需操作 |
| 上传中 | 正在上传音频 | 等待 |
| 处理中 | AI 正在分析 | 等待 |
| 已完成 | 分析完成 | 可在看板查看结果 |
| 失败 | 处理出错 | 点击「重试」按钮 |

## 离线支持

网络不好时录音会先保存在手机本地（最多 20 条），恢复网络后自动上传处理。进入录音页时也会自动检查并重试之前中断的录音。

## 常见问题

**Q: 录音按钮点不了？**
A: 检查是否已选择桌号。必须先选桌号才能录音。

**Q: 一直显示「处理中」？**
A: 正常处理时间为 10-15 秒。如果超过 1 分钟仍未完成，可能是网络问题，尝试刷新页面。如果状态变为「失败」，点击「重试」。

**Q: 隐身模式下怎么确认还在录音？**
A: 进入隐身模式前请注意录音已经开始（有波形动画）。退出隐身模式后可以看到计时器仍在运行。

**Q: 顾客说方言能识别吗？**
A: 可以。系统使用讯飞方言大模型，支持 200 多种方言自动识别，无需手动选择语言。
