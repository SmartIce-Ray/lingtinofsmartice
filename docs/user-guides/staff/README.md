# 店员使用手册

## 当前状态

店员专属功能正在规划中，目前尚未上线。

## 即将推出的功能

我们计划为服务员和厨师提供以下专属功能：

- **个人工作反馈**：查看与自己服务相关的顾客反馈和情绪评价
- **学习中心**：AI 生成的话术建议和服务技巧
- **任务通知**：接收店长分配的行动建议和待办事项
- **排班查看**：查看个人排班安排

## 现在可以做什么

在专属功能上线前，店员可以：
- 配合店长完成桌访——当店长进行桌访录音时，正常与顾客交流即可
- 参与例会——店长录制的例会纪要会在会后共享

## 联系我们

如果你有功能建议或需要帮助，请联系你的店长或管理员。
