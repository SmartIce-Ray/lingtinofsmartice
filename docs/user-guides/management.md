# 管理层使用手册

## 你的角色

作为管理层（老板、区域经理、运营总监），你关注**全局经营状况**——哪家门店执行到位、哪家顾客情绪异常、员工在关注什么问题。Lingtin 帮你用数据代替「走店凭感觉」。

## 六个核心页面

| 页面 | 路径 | 一句话说明 |
|------|------|-----------|
| **每日简报** | `/admin/briefing` | 登录后首页，跨门店异常问题清单，问题自带顾客原话 + 原声 |
| **经营看板** | `/admin/dashboard` | 一屏纵览所有门店的桌访执行和情绪指标 |
| **门店详情** | `/admin/restaurant/:id` | 深入查看单个门店的桌访记录和反馈 |
| **员工提问** | `/admin/staff-questions` | 了解各门店店长的 AI 问答和桌访内容 |
| **问卷管理** | `/admin/question-templates` | 配置桌访标准问题 |
| **AI 智库** | `/admin/chat` | 跨门店数据分析 |

## 典型使用场景

```
──── 早间巡查（5分钟）────
1. 打开每日简报，一眼看到哪些门店有问题需要关注
2. 问题卡片自带顾客原话，点「听原声」可播放录音
3. 点「查看详情」跳转到门店详情深入了解

──── 深入分析（按需）────
3. 查看具体桌访记录和顾客反馈原文
4. 在员工提问中了解店长关注点

──── 周度管理────
5. AI 智库提问：本周趋势
6. 更新问卷模板
```

## 每日简报（登录后首页）

登录后默认进入此页面。系统自动扫描所有门店昨日数据，检测异常并生成问题清单。

**异常检测规则：**
- 🔴 红色（严重）：菜品集中差评（同菜 ≥2 桌）、服务速度投诉（≥2 桌）、员工态度问题、情绪异常（< 0.5）
- 🟡 黄色（关注）：环境问题、覆盖率下降（< 70%）、桌访未执行、高优行动建议超 3 天未处理

**每个问题卡片包含：**
- 门店名称 + 问题类型 + 严重度标签
- 顾客原话（最多 3 条）
- 「听原声」按钮 — 播放原始桌访录音
- 「查看详情」— 跳转到门店详情页

**空状态：** 所有门店正常时显示「一切正常」+ 平均情绪/覆盖率统计

## 经营看板（多门店）

所有门店核心指标一屏总览：

- **汇总卡片**：门店总数、今日桌访总数、平均覆盖率、平均情绪分
- **关键词云**：跨门店顾客反馈关键词汇聚，词越大提及越多
- **门店网格卡片**：每家门店显示桌访数、覆盖率、平均情绪分，点击进入详情

---

## 门店详情

从经营看板点击门店卡片进入。展示该门店的桌访记录列表：

- 每条记录显示桌号、时间、情绪徽章（绿/灰/红）、AI 摘要
- 点击展开查看：转写全文、反馈详情、菜品提及

**使用建议：** 关注红色情绪徽章的记录，优先查看负面反馈详情。

---

## 员工提问

按门店和员工组织，展示各位店长的使用情况：

- AI 智库对话记录（店长问了什么、AI 怎么回答）
- 近期桌访记录摘要

**价值：** 了解团队关注点、评估使用深度、发现管理线索。多位店长重复问同一问题时，说明可能需要统一解决方案。

---

## 问卷管理

配置桌访标准问题模板，统一指导店长「今天重点问什么」。

**核心操作：**
- **查看/新建/编辑/删除模板**
- 每个模板包含模板名称、问题条目列表、生效日期、启用状态
- 保存后立即对店长生效

**最佳实践：**
- 问题控制在 3-5 个
- 定期根据数据反馈调整
- 新菜上市时必配问卷

---

## AI 智库（管理层版）

与店长版功能相同，但**数据范围覆盖所有门店**，可进行跨门店对比分析。

**管理层专属问题示例：**

| 场景 | 示例 |
|------|------|
| 门店对比 | 哪家门店本周覆盖率最高/最低？ |
| 全局趋势 | 所有门店这周平均情绪分比上周有变化吗？ |
| 菜品分析 | 哪道菜在多家门店同时收到差评？ |
| 员工表现 | 哪位店长的桌访覆盖率最高？ |
