# 管理层使用手册

## 你的角色

作为管理层（老板、区域经理、运营总监），你关注**全局经营状况**——哪家门店执行到位、哪家顾客情绪异常、员工在关注什么问题。Lingtin 帮你用数据代替「走店凭感觉」。

## 四个核心页面

| 页面 | 路径 | 一句话说明 |
|------|------|-----------|
| **总览** | `/admin/briefing` | 登录后首页——紧凑指标行 + 问题卡片 + 今日关键词 + 门店网格 |
| **顾客洞察** | `/admin/customer-insights` | 顾客建议（近 7 天跨门店）+ 反馈热词（高频差评/好评） |
| **产品洞察** | `/admin/product-insights` | 跨门店员工问题聚类，按关注人数排序 |
| **AI 智库** | `/admin/chat` | 跨门店数据分析，自然语言提问 |

> 问卷管理入口：点击页面右上角头像 → 选择「问卷管理」（路径：`/admin/question-templates/manage`）。

## 典型使用场景

```
──── 早间巡查（5分钟）────
1. 打开总览页，一眼看到指标行（门店数/桌访数/覆盖率/情绪分）
2. 查看问题卡片——哪些门店有异常
3. 点击卡片展开，看顾客原话 + Q&A 对话，听原声录音
4. 浏览今日关键词云，了解顾客高频反馈
5. 门店网格快速对比各店表现

──── 深入分析（按需）────
6. 切到「顾客洞察」查看近 7 天顾客建议趋势和反馈热词
7. 切到「产品洞察」看员工关注话题，发现跨门店共同问题

──── 周度管理 ────
8. AI 智库提问：本周趋势、跨门店对比
9. 从头像菜单进入问卷管理，更新标准问题
```

## 总览（登录后首页）

合并了原简报和经营看板，提供统一入口。

### 紧凑指标行

顶部一行四个核心数字：门店总数、今日桌访总数、平均覆盖率、平均情绪分。快速掌握全局状态。

### 问题卡片

系统自动扫描所有门店数据，检测异常并生成问题清单。

**异常检测规则：**
- 红色圆点（严重）：菜品集中差评（同菜 ≥2 桌）、服务速度投诉（≥2 桌）、员工态度问题、情绪异常（< 0.5）
- 黄色圆点（关注）：环境问题、覆盖率下降（< 70%）、桌访未执行、高优行动建议超 3 天未处理

**每个问题卡片包含：**
- 门店名称 + 问题类型 + 严重度圆点
- 顾客原话（最多 3 条）

**溯源操作：**
- 点击卡片展开 → 查看完整 Q&A 对话（店长提问 + 顾客回答）
- 点击播放按钮 → 听原始桌访录音

**空状态：** 所有门店正常时显示「一切正常」+ 平均情绪/覆盖率统计。

### 今日关键词

跨门店顾客反馈关键词汇聚，词越大提及越多。快速了解今天顾客最关注什么。

### 门店网格

每家门店一张卡片，显示桌访数、覆盖率、平均情绪分。点击进入门店详情页，查看该店的桌访记录和反馈。

门店详情页内容：
- 每条记录显示桌号、时间、情绪标签（绿/灰/红）、AI 摘要
- 点击展开查看：转写全文、反馈详情、菜品提及

## 顾客洞察

两个核心区块，帮你从顾客视角发现机会和问题。

### 顾客建议（近 7 天）

跨门店聚合顾客的建设性建议（如"建议加个冷面"、"希望有儿童套餐"）。AI 自动从桌访反馈中识别「建议」类型，区别于差评投诉。

- 按建议文本聚合，显示提及次数和来源门店
- 点击建议项展开 → 查看所有来源桌号 + Q&A 对话详情 + 播放原声录音

**使用建议：** 高频建议反映市场需求，可作为产品调整的参考依据。

### 反馈热词

高频差评和好评关键词分区展示。

- 差评热词区：高频出现的负面关键词，大小表示频率
- 好评热词区：高频出现的正面关键词
- 点击任意热词 → 展开查看相关对话详情 + 原声播放

**使用建议：** 差评热词是问题信号，好评热词是门店优势。

## 产品洞察

跨门店员工问题聚类分析，了解团队在关注什么。

**数据来源：** 员工 AI 智库对话 + 桌访内容。

**展示方式：**
- 自动归类为 7+1 个话题（如「菜品口味」「服务流程」「顾客投诉」等）
- 话题卡片按关注人数排序
- 3 家以上门店关注同一话题时标记「共同关注」标签

**使用建议：** 多位店长重复关注同一话题时，说明可能需要公司层面统一解决方案。

## 问卷管理

从页面右上角头像菜单进入「问卷管理」。配置桌访标准问题模板，统一指导店长「今天重点问什么」。

### 核心操作

- **查看 / 新建 / 编辑 / 删除模板**
- 每个模板包含模板名称、问题条目列表、生效日期、启用状态
- 保存后立即对店长生效

### 最佳实践

- 问题控制在 3-5 个
- 定期根据数据反馈调整
- 新菜上市时必配问卷

## AI 智库（管理层版）

与店长版功能相同，但**数据范围覆盖所有门店**，可进行跨门店对比分析。

**管理层专属问题示例：**

| 场景 | 示例 |
|------|------|
| 门店对比 | 哪家门店本周覆盖率最高/最低？ |
| 全局趋势 | 所有门店这周平均情绪分比上周有变化吗？ |
| 菜品分析 | 哪道菜在多家门店同时收到差评？ |
| 员工表现 | 哪位店长的桌访覆盖率最高？ |

## 常见问题

- **问卷管理入口在哪？** → 右上角头像菜单 → 问卷管理。
- **总览页没有问题卡片？** → 说明所有门店运营正常，会显示「一切正常」统计。
- **原来的经营看板去哪了？** → 已合并到总览页，门店网格和指标行提供相同信息。
- **怎么看单个门店详情？** → 总览页门店网格点击任意门店卡片进入。
