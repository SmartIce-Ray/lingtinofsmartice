# 门店详情

## 为什么需要门店详情

经营看板告诉你「哪家门店需要关注」，门店详情则让你深入了解「具体发生了什么」——查看每条桌访的顾客反馈原文、情绪评分和 AI 摘要，了解门店的真实经营状况。

## 前置条件

- 已登录管理员账号
- 从经营看板点击某个门店卡片进入，或直接访问对应 URL

## 页面内容

### 桌访记录列表

页面展示该门店的桌访录音记录，每条记录显示：

- **桌号**：如 A1、B3
- **时间**：录音时间
- **情绪徽章**：根据情绪分显示颜色标识
  - 绿色：正面（0.80-1.00）
  - 灰色：中性（0.40-0.79）
  - 红色：负面（0.00-0.39）
- **AI 摘要**：20 字以内的内容概述

### 展开查看详情

点击某条记录可以展开查看更多信息：

- **转写文本**：AI 纠偏后的完整对话文字记录
- **顾客反馈详情**：正面和负面反馈的具体内容
- **菜品提及**：提到了哪些菜品以及对应评价

## 使用场景

- **远程了解门店**：不用亲自到店，就能看到顾客的真实反馈
- **辅导店长**：发现店长话术问题或遗漏重要信息时，可以给出具体指导
- **追踪问题**：某道菜连续收到差评时，追溯最早是什么时候开始的

## 最佳实践

- **关注红色情绪徽章**——负面情绪的桌访优先展开查看，了解具体发生了什么
- **对比店长话术和顾客回答**——如果顾客回答很简短，可能是店长提问方式需要改进
- **结合行动建议**——在看到具体反馈后，回到经营看板检查该门店是否已有对应的行动建议

## 常见问题

**Q: 能看到录音音频吗？**
A: 当前版本主要展示文字转写内容。音频文件存储在云端，但界面上以文字形式呈现。

**Q: 记录多久会显示？**
A: 录音完成处理后（约 10-15 秒）即可在门店详情中看到。
