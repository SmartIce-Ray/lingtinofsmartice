# 经营看板

## 为什么需要多店看板

管理多家门店时，逐店巡查既费时又低效。经营看板让你在一个页面上看到所有门店的核心指标——谁在认真做桌访、哪家店顾客情绪异常、顾客最近都在关注什么——帮助你快速定位需要关注的门店，把时间花在最需要的地方。

## 前置条件

- 已登录管理员账号
- 旗下门店已有桌访录音数据

## 页面布局

登录后默认进入经营看板（`/admin/dashboard`），从上到下包含以下区域：

### 1. 汇总卡片

页面顶部显示全局汇总数据卡片，一眼了解整体情况：

- **门店总数**：当前管理的门店数量
- **今日桌访总数**：所有门店的桌访录音总计
- **平均覆盖率**：所有门店的平均桌访覆盖率
- **平均情绪分**：所有门店的平均顾客情绪

### 2. 关键词云

将所有门店的顾客反馈关键词汇聚在一起，以词云的形式展示。词越大代表被提及越多。帮助你发现跨门店的共性问题或亮点——比如多家门店同时出现「上菜慢」，可能是供应链或流程问题。

### 3. 门店网格卡片

每家门店显示为一张独立的卡片，包含：

- **门店名称**
- **今日桌访数**
- **覆盖率**（带颜色指示：绿色=良好、黄色=一般、红色=偏低）
- **平均情绪分**（带情绪指示颜色）

点击任意门店卡片，进入该门店的 [详情页](02-restaurant-detail.md)。

## 日期筛选

页面支持按日期查看数据，可以对比不同日期的整体表现。

## 最佳实践

- **每天花 2 分钟扫一眼**——快速识别覆盖率异常低或情绪异常差的门店
- **关注关键词云的变化**——如果某个负面关键词突然增大，说明多家门店出现了同一问题
- **定期对比日期**——观察各门店的趋势变化，而不只看单日数据

## 常见问题

**Q: 某家门店数据为空？**
A: 说明该门店当天没有已完成的桌访录音。可能是店长尚未使用系统，或录音正在处理中。

**Q: 覆盖率怎么算的？**
A: 桌访数 / 开台数。需要有开台数据才能计算。详见 [术语表](../getting-started/glossary.md#覆盖率)。
