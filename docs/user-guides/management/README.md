# 管理层使用手册

## 你的角色

作为管理层（老板、区域经理、运营总监），你关注的不是单桌的录音，而是**全局的经营状况**——哪家门店桌访执行到位、哪家门店顾客情绪异常、员工在问 AI 什么问题、桌访时应该问哪些标准问题。Lingtin 帮你用数据代替「走店凭感觉」，让管理决策有据可依。

## 五个核心页面

| 页面 | 路径 | 一句话说明 |
|------|------|-----------|
| **经营看板** | `/admin/dashboard` | 一屏纵览所有门店的桌访执行和情绪指标 |
| **门店详情** | `/admin/restaurant/:id` | 深入查看单个门店的桌访记录和顾客反馈 |
| **员工提问** | `/admin/staff-questions` | 了解各门店店长的 AI 问答和桌访内容 |
| **问卷管理** | `/admin/question-templates` | 配置桌访标准问题，指导店长桌访话术 |
| **AI 智库** | `/admin/chat` | 用自然语言提问，获取跨门店经营分析 |

页面之间通过顶部导航栏或侧边菜单切换。

## 典型使用场景

```
──── 早间巡查（5分钟）────
1. 打开经营看板，查看昨日各门店覆盖率和情绪概览
2. 发现某门店覆盖率偏低或负面情绪偏高 → 点击进入门店详情

──── 深入分析（按需）────
3. 在门店详情中查看具体桌访记录和顾客原始反馈
4. 在员工提问中了解店长在问 AI 什么——间接了解他们关注什么

──── 周度管理（每周一次）────
5. 在 AI 智库中提问：本周各门店覆盖率趋势、情绪变化
6. 更新问卷模板：根据本周数据，调整下周的桌访标准问题
```
