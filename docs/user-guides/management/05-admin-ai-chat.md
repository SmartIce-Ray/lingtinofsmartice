# AI 智库（管理层版）

## 与店长版的区别

管理层的 AI 智库和店长版功能相同，但**数据范围不同**——你可以查询所有门店的数据，而店长只能查询本店数据。这意味着你可以进行跨门店对比分析，发现全局性的趋势和问题。

## 前置条件

- 已登录管理员账号

## 页面入口

通过导航进入「AI 智库」页面（`/admin/chat`）。

## 使用方式

与店长版一致：在输入框中输入自然语言问题，AI 实时以打字机效果输出回答。支持快捷问题按钮和连续追问。

## 管理层专属问题示例

因为你能访问所有门店的数据，可以提出跨店对比和全局分析类的问题：

| 场景 | 问题示例 |
|------|----------|
| **门店对比** | 哪家门店本周覆盖率最高？最低的呢？ |
| **全局趋势** | 所有门店这周的平均情绪分是多少？比上周有变化吗？ |
| **菜品分析** | 哪道菜在多家门店同时收到差评？ |
| **问题排查** | XX 门店最近的投诉主要集中在什么方面？ |
| **员工表现** | 哪位店长的桌访覆盖率最高？ |
| **经营建议** | 基于最近的数据，你对整体经营有什么建议？ |

## 最佳实践

- **从全局到具体**——先问跨门店的整体情况，发现异常后再深入某家门店
- **用 AI 准备会议**——在管理会议前，用 AI 智库快速了解各门店近期数据，带着数据开会
- **定期做趋势分析**——每周问一次「本周各门店情绪变化趋势」，持续关注波动
- **结合问卷调整**——根据 AI 的分析结果，调整 [问卷管理](04-question-templates.md) 中的桌访标准问题

## 常见问题

**Q: AI 回答的数据范围是什么？**
A: 管理员账号可以查询所有门店的桌访记录、菜品提及、情绪数据等。

**Q: 能指定某家门店提问吗？**
A: 可以。在问题中指定门店名称，如「XX 店今天的覆盖率是多少」，AI 会自动筛选对应门店的数据。

**Q: 和店长看到的 AI 智库有什么不同？**
A: 界面和交互方式完全一样，唯一的区别是数据范围。管理员可以查询跨门店数据，店长只能查询本店数据。
